== Mock Tumblr

This app uses the Tumblr API and OAuth to mimick a users Tumblr account. User logs in using Tumblr OAuth and sees their current tumblr blog posts, followers, videos, etc. Any changes made to app will be reflected on Tumblr with ability to like posts, follow users, and share posts.
 * Cool feature:  In- app video streaming
 * Tech Stack:   Full api immersion, OAuth, JavaScript video streaming
